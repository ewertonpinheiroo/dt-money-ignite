# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/ewertonpinheiroo/dt-money-ignite.git
```

```bash
# Install Dependencies
$ Npm
# Run Aplication
$ Npm run dev
```
Go to http://localhost:5173/ to see the result.
